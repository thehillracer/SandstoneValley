# SandstoneValley
This is a plugin for levilamina that generates 512 by 512 plots infinitely. It also disables items such as pearls and crystals and has redstone kit commands, making it perfect for redstone building servers!

## Features:
• Generates plots infinitely.
• /kit command that provides essential redstone items.
• Assigns plots automatically.
• Disables features such as tnt minecarts and crystals.

## Requirements
• LegacyScriptingEngine quickjs
